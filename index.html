Validación de datos en Excel

¿Qué es?

La validación de datos es una función de excel que permie establecer que solo ciertos datos específicos puedan ser ingresados en una celda o grupo de celdas.De esta manera se limita el ingeso de números, decimales, fecha, hora y texto.

Para establecer la validación de datos se deben determinar directrices para establecer cantidades mínimas y máximas, omitir intervalos numéricos, fechas u horas, o limitar a cifras iguales, mayores o menores a un valor específico.

Además se pueden establecer las indicaciones o condiciones de los datos a ingresar con el mensaje de entrada, así mismo se puede describir porque no se permite el ingreso de ciertos datos con el mensaje de error.

Opciones de validación de datos en Excel:

  • Número entero: Se usa para valores numéricos cerrados, se aplica para el formato general y de número.
  • Decimal: Solo permite el ingreso de números decimales, se aplica para los formatos de número, porcentaje, contabilidad, moneda y general.
  • Lista: Establece un listado específico permitiendo el ingreso únicamente de los datos presentes en este listado ya sean datos númericos, texto o ambos.
  • Fecha: Delimita el ingreso de datos a solo fechas específicas ya sea en formato corto o largo.
  • Hora: Limita el ingreso de datos a solo horarios específicos.
  • Longitud de texto: Delimita el número de caracteres de texto que se pueden ingresar en una celda.
  • Perzonalizada: Permite establecer fórmulas particulares para delimitar el ingreso de los datos.

Condiciones en la validación de datos:

  • Entre: Establece un intervalo con una cigra mínima y una cifra máxima.
  • No está entre: Omite un intervalo, se indica dónde empieza y dónde termina el intervalo.
  • Igual a: Permite el ingreso de un único valor.
  • No igual a: Descarta un valor específico.
  • Mayor que: Permite únicamente valores superiores al indicado.
  • Menor que: Permite únicamente valores inferiores al indicado.
  • Mayor o igual a que: Acepta valores iguales o superiores al valor indicado.
  • Menor o igual a que: Acepta valores iguales o inferiores al valor indicado.

¿Cómo realizar la validación de datos?

  1. Seleccionar las celdas:Para explicar los pasos de la validación se usará el siguiente ejemplo. En un restaurante pequeño cada semana se realiza la compra de los ingredientes con menos stock, pero el chef ejecutivo estableció que no se pueden comprar mas de 30 unidades de cada ingrediente, para ello se usará una validación de datos para las unidades a comprar. Para esto se inicia seleccionando las celdas para aplicar la validación.
  2. Hacer clic en "Validación de datos": En el menú superior encontrarás la pestaña de "Datos", allí se encuntra el icono de una tabla con un símbolo de check verde que es el de validación de datos, al dar click ahí se abrirá un recuadro.
  3. Configurar el tipo de datos: En la pestaña de configuración del nuevo recuadro seleccionar "Permitir" se desplegará una lista en la que seleccionaremos para este ejemplo la opción de "Número entero".

    Luego aparecerán nuevas opciones en donde seleccionaremos el rango o condición a utilizar, en este caso "menor o igual que". A continuación indicaremos el valor de referencia, en este ejemplo "30".
  4. Establecer el "mensaje de entrada": En el recaudro seleccionar la pestaña de "mensaje de entrada", allí aparecerán dos espacios, uno pide el título y el otro pide el mensaje de entrada, este mensaje aparecerá cada vez que se de clic en una de las celdas seleccionadas. El mensaje debe contener instrucciones claras y sencillas.
  5. Establecer el "mensaje de error": Lo último por completar es la pestaña de "mensaje de error", allí se redactará un breve mensaje para aquellos que ingresen los datos de forma incorrecta. En la pestaña de "Estilo" aparecen 3 opciones, "alto" que no permitirán de ninguna forma ingresar datos que no cumplan con lo establecido en la validación de datos, "advertencia" y "información" que notificarán que hay un error en los datos, pero sí dejará ingresar los datos a la celda. Al igual que en el mensaje de entrada el mensaje debe ser claro y sencillo.

    Al finalizar estos pasos damos clic en "Aceptar".
  6. Verificar la validación: El último paso es comprobar que la validación qeudó aplicada correctamente. Para esto ingresaremos que no se ajuste a las condiciones dadas, para el ejemplo, un dato mayor a 30, inmediatamente arrojará un recuadro con el mensaje de error.
